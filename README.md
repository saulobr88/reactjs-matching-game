# Matching Game

[![Netlify Status](https://api.netlify.com/api/v1/badges/cffa7738-1a90-40e6-af3b-9d404bd5aa00/deploy-status)](https://app.netlify.com/projects/matching-game-s/deploys)

It is just a simple and fun matching people profile game, users enter with name and birthdate and get a score, this score could be used to compare and making fun about themselves

### the project structure

this is based on ViteJs (React + Typescript template)

## Installation

If you would like to download the code and try it for yourself:

1. git clone `this repo`
2. cd `this repo name`
3. pnpm install / npm install
4. pnpm run dev / npm run dev

And have fun
